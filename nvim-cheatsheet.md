# Neovim チートシート (LazyVim)

> 迷ったときの索引。キーは Normal モード基準（別途記載あり）。

---

## 移動

### 基本

| キー | 動作 |
|------|------|
| `h` `j` `k` `l` | 左・下・上・右 |
| `w` / `b` | 次/前の単語先頭 |
| `e` / `ge` | 次/前の単語末尾 |
| `W` / `B` | 空白区切りの次/前の単語 |
| `0` | 行頭（文字なし） |
| `^` | 行頭の最初の非空白文字 |
| `$` | 行末 |
| `g_` | 行末の最後の非空白文字 |
| `gg` | ファイル先頭 |
| `G` | ファイル末尾 |
| `{N}G` | N行目へ |
| `%` | 対応する括弧へ |

### 画面

| キー | 動作 |
|------|------|
| `<C-d>` / `<C-u>` | 半ページ下/上スクロール |
| `<C-f>` / `<C-b>` | 1ページ下/上スクロール |
| `zz` | カーソル行を画面中央に |
| `zt` / `zb` | カーソル行を画面上/下端に |

### ジャンプ

| キー | 動作 |
|------|------|
| `<C-o>` | ジャンプ履歴を戻る |
| `<C-i>` | ジャンプ履歴を進む |
| `''` | 直前のジャンプ位置に戻る |
| `m{a-z}` | マーク設定 |
| `` `{a-z} `` | マーク位置へジャンプ |

---

## 編集

### 挿入モードに入る

| キー | 動作 |
|------|------|
| `i` / `I` | カーソル前 / 行頭に挿入 |
| `a` / `A` | カーソル後 / 行末に挿入 |
| `o` / `O` | 下/上に新行を挿入 |
| `s` | 1文字削除して挿入 |
| `S` | 行全体を削除して挿入 |
| `C` | カーソルから行末まで削除して挿入 |

### 削除・変更

| キー | 動作 |
|------|------|
| `x` | 1文字削除 |
| `dd` | 行削除 |
| `D` | カーソルから行末まで削除 |
| `dw` / `db` | 次/前の単語を削除 |
| `diw` | 単語を削除（空白含まず） |
| `daw` | 単語を削除（空白含む） |
| `di"` / `da"` | `"` 内を削除（括弧も同様） |
| `cc` | 行全体を変更 |
| `cw` | 単語を変更 |
| `ciw` | 単語を変更（空白含まず） |
| `ci"` | `"` 内を変更 |
| `r{c}` | 1文字を `c` に置換 |
| `u` | アンドゥ |
| `<C-r>` | リドゥ |

### コピー・ペースト

| キー | 動作 |
|------|------|
| `yy` | 行をヤンク（コピー） |
| `yw` / `yiw` | 単語をヤンク |
| `p` / `P` | カーソルの後/前にペースト |
| `"+y` | システムクリップボードにコピー |
| `"+p` | システムクリップボードからペースト |

---

## 検索・置換

| キー / コマンド | 動作 |
|------|------|
| `/{pattern}` | 前方検索 |
| `?{pattern}` | 後方検索 |
| `n` / `N` | 次/前のマッチへ |
| `*` / `#` | カーソル単語を前方/後方検索 |
| `:%s/old/new/g` | ファイル全体で置換 |
| `:%s/old/new/gc` | 確認しながら置換 |
| `:'<,'>s/old/new/g` | 選択範囲で置換 |

---

## ビジュアルモード

| キー | 動作 |
|------|------|
| `v` | 文字選択 |
| `V` | 行選択 |
| `<C-v>` | 矩形選択 |
| `gv` | 直前の選択を再選択 |
| `>` / `<` | インデント増/減 |
| `=` | 自動インデント |

---

## ウィンドウ・バッファ・タブ

### ウィンドウ

| キー | 動作 |
|------|------|
| `<C-w>s` / `<C-w>v` | 水平/垂直分割 |
| `<A-h>` `<A-j>` `<A-k>` `<A-l>` | ウィンドウ移動（カスタム） |
| `<C-w>=` | ウィンドウサイズを均等に |
| `<C-w>q` | ウィンドウを閉じる |

### バッファ

| キー | 動作 |
|------|------|
| `<S-h>` / `<S-l>` | 前/次のバッファ |
| `<leader>bd` | バッファを閉じる |
| `<leader>bb` | 直前のバッファに戻る |

### タブ

| キー | 動作 |
|------|------|
| `<leader><tab>n` | 新しいタブ |
| `<leader><tab>d` | タブを閉じる |
| `<leader><tab>]` / `<leader><tab>[` | 次/前のタブ |

## LSP（コード インテリジェンス）

| キー | 動作 |
|------|------|
| `gd` | 定義へジャンプ |
| `gD` | 宣言へジャンプ |
| `gr` | 参照一覧 |
| `gI` | 実装へジャンプ |
| `gy` | 型定義へジャンプ |
| `K` | ホバードキュメント |
| `<C-t>` | ジャンプ前に戻る |
| `<leader>cr` | リネーム |
| `<leader>ca` | コードアクション |
| `<leader>cf` | フォーマット |

## 診断（エラー・警告）

| キー | 動作 |
|------|------|
| `]d` / `[d` | 次/前の diagnostic へ |
| `]e` / `[e` | 次/前の **Error** へ |
| `]w` / `[w` | 次/前の **Warning** へ |
| `<leader>xx` | Trouble: ファイル内一覧 |
| `<leader>xX` | Trouble: ワークスペース一覧 |
| `<leader>cs` | シンボル一覧 (Trouble) |

## ファイル検索・grep（Telescope / Snacks）

| キー | 動作 |
|------|------|
| `<leader><space>` | ファイル検索 |
| `<leader>ff` | ファイル検索 (cwd) |
| `<leader>fg` | Git ファイル検索 |
| `<leader>fr` | 最近のファイル |
| `<leader>/` | ライブ grep |
| `<leader>sg` | grep 検索 |
| `<leader>sw` | カーソル単語で grep |

## Git

| キー | 動作 |
|------|------|
| `<leader>gg` | Lazygit を開く (cwd) |
| `<leader>gG` | Lazygit を開く (ファイルのディレクトリ) |
| `<leader>gl` | Lazygit ログ (cwd) |
| `<leader>gL` | Lazygit ログ (ファイルのディレクトリ) |
| `<leader>gd` | Git diff |
| `<leader>gb` | Git blame |
| `]h` / `[h` | 次/前の hunk へ |
| `<leader>ghs` | hunk をステージ |
| `<leader>ghr` | hunk をリセット |
| `<leader>ghp` | hunk をプレビュー |

## その他便利操作

| キー | 動作 |
|------|------|
| `<leader>e` | ファイルエクスプローラー (Neo-tree) |
| `<leader>E` | ファイルエクスプローラー (cwd) |
| `<leader>-` | スプリットウィンドウでファイル開く |
| `gcc` | 行コメントトグル |
| `gc{motion}` | 範囲コメントトグル |
| `<leader>ui` | インデントガイド表示 |
| `<leader>uw` | 折り返し表示トグル |
| `<leader>un` | 行番号トグル |
| `za` | フォールドトグル |
| `zR` / `zM` | 全フォールドを開く/閉じる |
| `.` | 直前の操作を繰り返す |
| `q{a}` / `@{a}` | マクロ記録/再生 |
| `@@` | 直前のマクロを再実行 |
| `<leader>L` | LazyVim の情報 |
| `<leader>l` | Lazy プラグイン管理 |
